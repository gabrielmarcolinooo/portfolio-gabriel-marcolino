# 🤖 Identificação de Viés em Modelos de Inteligência Artificial

## 📝 Descrição do Projeto
Este projeto consiste em uma análise prática sobre viés em modelos de Inteligência Artificial aplicados à classificação de imagens. O objetivo principal foi identificar falhas de interpretação realizadas pelo modelo durante o processo de reconhecimento visual, além de compreender os impactos sociais causados por classificações incorretas feitas pela IA.

Desenvolvido como atividade acadêmica do curso de **Análise e Desenvolvimento de Sistemas**, o projeto demonstra um exemplo onde o modelo classificou incorretamente uma pessoa como “Atleta”, quando deveria ser identificada como “Sedentária”. A atividade também aborda mecanismos de viés, consequências sociais e possíveis ações mitigadoras para reduzir erros em sistemas inteligentes.

*Figura 1: Evidência de erro de classificação realizada pelo modelo de IA.*

## 🚀 Tecnologias Utilizadas
* **Área:** Inteligência Artificial
* **Técnica:** Classificação de Imagens
* **Ferramentas:** Plataforma de treinamento de IA
* **Conceitos:** Machine Learning, Viés Algorítmico
* **Recursos:** Treinamento e validação de modelos

## 📊 Resultados e Aprendizados
O projeto permitiu compreender como modelos de IA podem apresentar vieses durante classificações automáticas.

* **Identificação de Viés:** O sistema classificou incorretamente uma pessoa parada como “Atleta”.
* **Impacto Social:** Foi possível analisar como erros de IA podem reforçar estereótipos e julgamentos baseados apenas na aparência.
* **Consciência Ética:** Aprendi a importância da qualidade dos dados utilizados no treinamento dos modelos.
* **Mitigação de Erros:** O projeto demonstrou que a revisão humana e validação dos dados podem reduzir falhas e vieses algorítmicos.

## 🔧 Como Executar
1. Acesse a plataforma de treinamento do modelo.
2. Realize o upload das imagens para cada categoria.
3. Treine o modelo utilizando os dados fornecidos.
4. Teste novas imagens e analise os resultados da classificação.
5. Avalie possíveis erros e vieses identificados pelo sistema.

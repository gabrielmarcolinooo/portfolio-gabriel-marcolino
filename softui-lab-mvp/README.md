# 🎨 SoftUI Lab — MVP de Gerador Neumórfico

## 📝 Descrição do Projeto
Este projeto consiste no desenvolvimento de um MVP de uma aplicação web chamada **SoftUI Lab**, criada para gerar designs neumórficos profissionais de forma interativa. O objetivo principal é permitir que o usuário personalize elementos visuais, ajuste parâmetros de estilo e visualize o resultado em tempo real dentro da própria interface.

Desenvolvido como parte das atividades de **Engenharia Reversa, Prototipagem e Aplicações com IA**, o projeto evoluiu a partir da recriação de uma interface inspirada no Neumorphism.io, adicionando melhorias como navegação, controles visuais, geração aleatória, salvamento de designs, autenticação e persistência de dados.

## 🚀 Tecnologias Utilizadas
* **Linguagens:** HTML5, CSS3 e JavaScript
* **Conceitos:** MVP, Engenharia Reversa, Soft UI e Neumorphism
* **Funcionalidades:** Gerador visual, salvamento de designs e autenticação
* **Ferramentas:** Google AI Studio, VS Code e navegador web
* **Design:** Interface responsiva, moderna e minimalista

## 📊 Resultados e Aprendizados
O projeto apresentou uma evolução significativa em relação ao clone inicial, transformando a interface em um MVP mais completo e funcional.

* **MVP Funcional:** A aplicação passou a contar com recursos além da geração visual básica.
* **Interatividade:** Foram adicionados controles para personalização dos elementos neumórficos.
* **Persistência de Dados:** Implementação de salvamento e carregamento de designs do usuário.
* **Melhoria de Interface:** A experiência visual foi refinada com layout mais profissional e organizado.
* **Correção de Bugs:** Foram ajustados problemas relacionados ao upload de imagens, salvamento de estado e funcionamento dos controles.

## 🔧 Como Executar
1. Clone o repositório.
2. Abra o projeto no VS Code.
3. Execute o arquivo `index.html` no navegador.
4. Utilize o gerador para personalizar os elementos neumórficos.
5. Teste as funções de gerar, salvar e carregar designs.
